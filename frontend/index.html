Code — Feature Extraction and Anomaly Detection

The core algorithmic logic powering SkySecure V2. Five independent modules, each unit-tested, following SOLID design principles and deployed via Docker containers.

feature_extractor.py

What This Does

  • Reads each incoming ADS-B frame from the data stream
  • Computes 14 features per frame including velocity delta, altitude rate, lat/lon displacement
  • Checks velocity, altitude, lat/lon delta, and ICAO novelty against known baselines
  • Returns a normalised feature vector as input to the ML pipeline
  • Flags physically impossible readings immediately before ML inference
import numpy as np from sklearn.ensemble import IsolationForest from math import radians, sin, cos, sqrt, atan2 def haversine(lat0, lon0, lat1, lon1): R = 6371000 phi0, phi1 = radians(lat0), radians(lat1) dphi = radians(lat1 - lat0) dlambda = radians(lon1 - lon0) a = sin(dphi/2)**2 + cos(phi0)*cos(phi1)*sin(dlambda/2)**2 return R * 2 * atan2(sqrt(a), sqrt(1-a)) class FeatureExtractor: def extract(self, frame, history): features = {} features['lat'] = frame['lat'] features['lon'] = frame['lon'] features['altitude'] = frame['altitude'] features['speed'] = frame['speed'] features['heading'] = frame['heading'] if len(history) >= 2: t0, lat0, lon0 = history[-2] t1, lat1, lon1 = history[-1] dt = t1 - t0 if dt > 0: dist = haversine(lat0, lon0, lat1, lon1) speed_ms = dist / dt if speed_ms > 650: # extreme positional jump features['movement_anomaly'] = True features['anomaly_speed_ms'] = speed_ms return np.array(list(features.values()), dtype=np.float32) def _velocity_check(self, frame): if frame['speed'] > 850: return 10.0 # max anomaly score return 0.0

anomaly_classifier.py — Spoof probability scoring

What This Does

  • Isolation Forest partitions the feature space to isolate anomalies from normal data clusters
  • LSTM flags physically implausible state transitions such as 900 kts at 200 ft altitude
  • Ensemble score: 40% Isolation Forest weight + 60% LSTM weight
  • Alert triggered when score is >= 5 for >= 3 consecutive frames
  • Confidence intervals output alongside each threat score
def heuristic_spoof_probability(df): prob = pd.Series(0.0, index=df.index) # Impossible coordinates — outside Earth bounds prob += np.where( (df['Latitude'].abs() > 90) | (df['Longitude'].abs() > 180), 1.0, 0 ) # Sentinel altitude value used in spoofed datasets prob += np.where(df['Altitude_ft'] == 999999, 0.95, 0) # Physically impossible speed prob += np.where(df['Speed_knots'] > 1200, 0.85, 0) # Negative speed value prob += np.where(df['Speed_knots'] < 0, 0.9, 0) return prob.clip(0, 1) # Add movement anomaly contribution df['Spoof_Probability'] = heuristic_spoof_probability(df) df['Spoof_Probability'] += np.where(df['Movement_Anomaly'], 0.3, 0) df['Spoof_Probability'] = df['Spoof_Probability'].clip(0, 1)

ml_engine.py — LSTM trajectory model

What This Does

  • Two-layer LSTM with hidden size 64 trained on ADS-B trajectory sequences
  • Input: sequence of 20 frames with lat, lon, altitude, speed, heading
  • Predicts next state — large prediction error indicates anomalous trajectory
  • Ensemble: 60% LSTM weight + 40% Isolation Forest weight
import torch import torch.nn as nn class TrajectoryLSTM(nn.Module): def __init__(self, input_size=5, hidden_size=64, num_layers=2): super().__init__() self.lstm = nn.LSTM( input_size, hidden_size, num_layers, batch_first=True, dropout=0.1 ) self.fc = nn.Linear(hidden_size, input_size) def forward(self, x): out, _ = self.lstm(x) return self.fc(out[:, -1, :]) def ensemble_score(if_score, lstm_score): return (0.4 * if_score) + (0.6 * lstm_score) # Alert: score >= 5 for >= 3 consecutive frames if len(score_buffer) >= 3: if all(s >= 5 for s in score_buffer[-3:]): alert_dispatcher.trigger(aircraft_id, score_buffer[-1])
